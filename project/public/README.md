Place your custom images here so Vite serves them at the project root.

Files to add:
- logo.png — top-left logo (recommended: 64x64px, square, PNG with transparency)
- bck.png  — page background (recommended: 1920x1080px, high quality)

After adding images, restart the dev server or refresh the page:

npm run dev

They will be available at `/logo.png` and `/bck.png` in the app.
